## Graph Class - Algorithm Coverage

The Graph class implements a comprehensive set of algorithms:

TRAVERSAL          SHORTEST PATH      MST ALGORITHMS     DAG OPERATIONS
├─ BFS()           ├─ Dijkstra()       ├─ PrimMST()        └─ TopologicalSort()
└─ DFS()           └─ BellmanFord()    └─ KruskalMST()

This single class covers ~50% of the project's algorithmic requirements!
